## 1. 결과물 및 객관적 증거

### OOM
- [ ] 메모리 사용량의 지속 증가와 종료 시점이 기록되어 있는가?
- [ ] `MEMORY_LIMIT` 변경 전후 생존 시간과 메모리 수치를 비교했는가?

### CPU
- [ ] 특정 프로세스의 CPU 급상승과 종료 로그가 기록되어 있는가?
- [ ] `CPU_MAX_OCCUPY` 변경 전후 종료 여부 또는 생존 시간을 비교했는가?

### Deadlock
- [ ] PID는 존재하지만 CPU·메모리·로그가 정체된 상태를 식별했는가?
- [ ] `MULTI_THREAD_ENABLE` 변경 전후 재현·회피 결과를 비교했는가?

### 리포트 공통
- [ ] 3건 모두 현상 → 증거 → 원인 → 조치 → 검증 구조인가?
- [ ] PID, 측정 시각, 명령어, 핵심 로그, 스크린샷이 포함되어 있는가?
- [ ] 예시 데이터와 실제 측정 데이터를 명확히 구분했는가?
